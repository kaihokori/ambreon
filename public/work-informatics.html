Informatics Analysis

Tools for exploring personal activity data while keeping privacy protected.

Understand activity patterns with privacy in mind

These tools clean and summarise minute-level activity data and create easy-to-read visuals. Outputs focus on general patterns and do not include sensitive individual details.

Focus

The work centres on getting minute-by-minute activity data into a clean form, deciding sensible thresholds for activity levels, and creating simple visuals that explain common patterns without revealing any one person's data.

What you'll get

Data pipelines

Automated steps that clean and organise activity data so it's ready to use.

Interactive notebooks

Interactive reports that walk through the analysis and show non-sensitive charts and summaries.

Privacy-first reporting

Grouped summaries and visuals that keep individual data private while still showing useful trends.

Approach

  • Apply clear checks to clean and validate the data.
  • Use straightforward rules to group activity into easy-to-understand levels.
  • Share only aggregated visuals and avoid raw or identifying data in public summaries.

How it's done

Data processing

Standard tools that clean and organise activity records.

Visuals

Interactive and static charts to explore daily and weekly patterns.

Reports

Clear, documented reports that explain the steps taken and the findings in plain language.

Privacy

To protect privacy, public summaries do not include any identifying details or raw data. Detailed records are kept secure and only shared under clear, agreed terms.
